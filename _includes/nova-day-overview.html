Nova STEM Day - May 5, 2019 at TBD (Sammamish or Redmond)

It can be difficult to get started on a Nova award. Where should you start? What should you do?.

Nova STEM Day has been created to help you out. In this very hands-on day, Scouts will participate in activities in each category - Science, Technology, Engineering and Math. All of the activities are part of current Nova modules, so you as the adult leader can also get some ideas (and we'll have a special session just for adult leaders who want to learn more about the Nova program and how to get started).

Check out all of the activities Scouts will participate in throughout the day.

Science

Learn how to use the scientific method in a hands-on experiment. In this activity you will learn about chemical reactions while using the scientific method - question, hypothesize, experiment, observe and record, analyze and share results. You will don a lab coat and googles and see what happens when you mix an acid and a powder.

This activity is part of the Science Everywhere! Nova Award module.

Technology

Learn how to write a computer program to control a robot. In this hands-on activity, Scouts will learn the basic or writing a software program to control a small robot. They will experiment with different methods and conditions in an easy-to-use, kid-friendly programming tool. No experience necessary.

This activity is part of the Cub Scouts Can Code Nova Award module.

Engineering

In this exciting activity Scouts will experiment with a Pinewood Derby car to learn about physics. THrough some experiments the Scouts will learn what simple modifications improve the physics of a Pinewood car, making it go faster. They will race their cars down a real Pinewood Derby track several times to see what effect their changes have.

This activity is part of the Swing! Nova Award module.

Math

Learn how to be a spy or a hacker in this code creating, code breaking activity. During this hands-on activity Scouts will use different techniques to write messages in code and share them with others. Can they break someone elses code?. At the end of the day there will be a raffle - you have to crack the code to enter.

This activity is part of the 1-2-3 Go! Nova Award module.



More Information

Nova STEM Day is a free introduction to the Nova Award program, put on by Cub Scout Pack 571 (Sammamish). The goal for the day is to get as many Cub Scouts as possible excited about STEM and the Nova program, and to help adult leaders who are interested to learn how to start a Nova program in their unit.
